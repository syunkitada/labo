# frr2

- 3 段 CLOS 構成
- spine は 2 台で 1 セット、leaf も 2 台で１セットのアクティブアクティブ構成
- 各 spine は配下の各 leaf へそれぞれ配線される
- 各 leaf は配下の各 HV へそれぞれ配線される

## 構成

```
spine1[1-2] --- leaf1[1-2] --- hv111
                           --- hv112
                           ...
            --- leaf2[1-2] --- hv121
                           --- hv122
                           ...

```
